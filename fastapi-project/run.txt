uvicorn fastapi-project.main:app --reload  #서버 실행 코드

1. 모든 강의 정보를 가져오는 엔드포인트
http://127.0.0.1:8000/courses


2. 사용자의 전공에 관한 모든 과목을 가져오는 엔드포인트 예시
http://127.0.0.1:8000/courses/소프트웨어공


3. 사용자의 전공, 학년, 이수구분을 입력하여 필터링하는 엔드포인트 예시
http://127.0.0.1:8000/courses/소프트웨어공/3/전공선택
http://127.0.0.1:8000/courses/소프트웨어공/3/전공필수 
http://127.0.0.1:8000/courses/컴퓨터공학과/4/전공선택


4. 사용자가 원하는 시간, 건물에 모든 강의를 필터링하는 엔드포인트 예시
http://127.0.0.1:8000/courses/월 1/공과대학 3호관


5. 사용자의 전공, 원하는 시간대, 교양or일반선택을 입력하면 강의평점이 있는 강의들에서 상위 5개 추천
http://127.0.0.1:8000/recommend_courses/소프트웨어공/월 1/일반선택
